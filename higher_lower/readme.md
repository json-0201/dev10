<h2 align="center">Introduction</h2>
<p align="center">In this exercise, you will develop a number guessing game.</p>
<h2 align="center">Instructions</h2>
<p align="center">Create a webpage that simulates a number guessing game. On load, the page should generate a random number between 1 and 20. The user will enter a guess into a text input and click a button to submit the guess.</p>
<p align="center">When submitted, one of the following messages should be displayed:</p>
<p align="center">- "You got it!" if the guess is right.</p>
<p align="center">- "No, try a higher number." if the guess is too low.</p>
<p align="center">- "No, try a lower number." if the guess is too high.</p>
